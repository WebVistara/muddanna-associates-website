# 🏢 Muddanna Associates Website

A modern, responsive multi-page website developed for **Muddanna Associates** to showcase their services, portfolio, and company profile in a professional and engaging way.

---

## 🚀 Features

* 🌐 Multi-page layout (Home, About, Services, Clients, Gallery, Contact)
* 📱 Fully responsive design for all devices
* ⚡ Optimized images (`.webp`, `.avif`) for fast performance
* 🧾 Clean and professional business UI
* 📷 Gallery section for showcasing projects/work
* 📞 Contact page for inquiries and communication
